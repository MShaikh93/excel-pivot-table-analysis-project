# Excel Pivot Table Analysis Project

## Project Overview

This project demonstrates practical use of Microsoft Excel Pivot Tables for business reporting and data analysis using the Superstore Sales Dataset.

The workbook covers key Pivot Table concepts commonly used by Data Analysts, Business Analysts, and reporting professionals.

---

## Skills Demonstrated

* Create Pivot Tables
* Rows, Columns, Values & Filters
* Sort & Filter Pivot Data
* Group Dates
* Value Field Settings
* Show Values As (% of Total)
* Pivot Charts
* Slicers
* Timeline Filters
* Calculated Fields
* KPI Reporting

---

## Dataset

Superstore Sales Dataset

---

## Workbook Structure

### 01_Sales

Basic Sales Summary using Pivot Tables

### 02_Region

Regional Segment Analysis using Rows, Columns, Values, and Filters

### 03_State

State Sales Ranking using Sorting and Filtering

### 04_Trend

Monthly Sales Trend Analysis using Date Grouping

### 05_Values

Value Field Settings (Sum, Average, Min, Max, Count)

### 06_Percent

Percentage Contribution Analysis

### 07_Charts

Pivot Chart Creation and Trend Visualization

### 08_Slicers

Interactive Reporting using Slicers and Timelines

### 09_Calc

Calculated Fields and Profit Margin Analysis

### 10_KPI_Report

Executive KPI Reporting and Business Performance Summary

## Key Business Insights

* Technology generated the highest sales and profit among all categories.
* The West region delivered the strongest overall business performance.
* Office Supplies and Technology achieved the highest profit margins.
* Furniture generated strong sales but lower profitability.
* Overall business operations achieved an estimated profit margin of 12%.
